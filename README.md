# GraphNetExtractAgent

本项目基于 [GraphNet](https://github.com/PaddlePaddle/GraphNet) 仓库构建，遵循其贡献指南和教程。

## 项目背景

该项目是根据 GraphNet 仓库中的 `CONTRIBUTE_TUTORIAL.md` 构建的，用于模型计算图的自动提取和管理。

## 参考文档

- [贡献教程](./docs/guides/CONTRIBUTE_TUTORIAL.md) - 详细的模型提取和提交流程
- [贡献教程（中文）](./docs/guides/CONTRIBUTE_TUTORIAL_cn.md) - 中文版本贡献指南
- [快速开始](./docs/QUICK_START.md) - 项目快速入门指南
- [HuggingFace 模型修复指南](./docs/huggingface_model_repair_guide.md) - 模型修复最佳实践

## 目录结构

- `docs/` - 文档和指南
- `scripts/` - 提取脚本
- `data/` - 数据文件
- `results/` - 提取结果
- `task_tracking/` - 任务跟踪
- `huggingface/` - HuggingFace 相关工具
